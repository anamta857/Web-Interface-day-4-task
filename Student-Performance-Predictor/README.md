# Student Performance Predictor

A responsive web application built using React.js to predict a student's academic performance based on study-related factors.

## Features

- Responsive User Interface
- Input Fields:
  - Study Hours
  - Attendance
  - Assignments Completed
  - Previous Marks
- Predicts Student Grade
- Displays Performance Category

## Performance Categories

- Excellent
- Good
- Average
- Needs Improvement

## Technologies Used

- React.js
- JavaScript
- CSS
- Vite

## How to Run

1. Clone the repository

```bash
git clone <repository-url>
Install dependencies
npm install
Start the development server
npm run dev
Project Objective

The purpose of this project is to provide a simple prediction system that evaluates student performance using academic indicators and displays the expected grade and performance category.
